# https://in-info-web4.luddy.indianapolis.iu.edu/~jat5/final-project/

## Description

Final project for N215 Front End Development.  
This project contains a responsive multi-page/site-style frontend implementation, including a styled contact section built with SCSS.

## Project Structure

```text
final-project/
├─ scss/
│  └─ contact.scss
├─ css/
│  └─ ...compiled styles...
├─ js/
│  └─ ...scripts...
├─ assets/
│  └─ ...images/fonts/media...
├─ index.html
└─ readme.md
```

## Dependencies

- **Node.js** (recommended for local tooling)
- **Sass (Dart Sass)** for compiling `.scss` to `.css`
  - Install: `npm install -D sass`
  - Example compile command: `npx sass scss:css --watch`
- Optional: **Live Server** (or similar) for local development preview
